## 2. Current Review Process Description
- Receives batch of pre-screened applications
- Initial scanning pass to get overall sense
- Reads applications in smaller batches
- Reviews individually, comparing against rubric
- Iterative scoring process, revisiting and adjusting scores
- Provides detailed feedback for applicants

## 3. Pain Points & Challenges
- Rubrics can feel restrictive
- Difficulty standardizing "value" across different perspectives
- Challenge of evaluating proposals across different disciplines
- Managing time and maintaining focus during reviews

## 4. What They Value in Applications
- Motivation letters showing proactive attitude
- Realistic, achievable project proposals

- Reference letters with substantive, personalized content
- Candidates demonstrating potential despite circumstances
