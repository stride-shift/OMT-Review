## 2. Current Review Process Description
- Initial 5-minute quick scan of applicants
- Detailed review: 45-60 minutes per application over 2-3 evenings
- Comprehensive document review
- Systematic approach checking:
  - Consistency of documents
  - Academic departments
  - Broader application narrative
- Final 20-minute consistency check across all applications
- Uses a 4-5 point scoring rubric
- Writes 2-3 sentence rationale for scoring

## 3. Pain Points & Challenges

- Balancing individual context with objective assessment

## 4. What They Value in Applications

- Contextual understanding of applicant's background
